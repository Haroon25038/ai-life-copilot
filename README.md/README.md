# AI Life Copilot

This project is a machine learning-based system that predicts student performance and provides personalized recommendations.

## Features

* Student performance prediction
* AI-based suggestions
* Streamlit dashboard

## Technologies Used

* Python
* Scikit-learn
* Pandas
* Streamlit

## How to Run

1. Install dependencies:
   pip install -r requirements.txt

2. Run the app:
   streamlit run app/app.py
